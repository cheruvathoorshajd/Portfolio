## 🔄 Easy to Update

### Change Projects
Edit `src/constants.js`:
```javascript
export const PROJECTS = [
  {
    id: 1,
    title: 'New Project',
    category: 'Category',
    image: '/image.png',
    link: 'https://...'
  }
];
```

### Change Skills
Edit `src/constants.js`:
```javascript
export const TECHNICAL_SKILLS = ['Skill 1', 'Skill 2', ...];
```

### Change Colors
Edit `src/styles/index.css`:
```css
:root {
  --accent: #ff3c00; /* Your primary color */
}
```
